z390 GUI Interface Demo
The screen capture below shows 2 copies of the z390 GUI interface running side by side.  The window on the left shows the output trace of demo program demowto1.mlc after having selected ASMLG on the file menu and selecting the demo\demowto1.mlc file to assemble, link, and execute.  The copy on the right shows the file menu drop down menu with ASMLG and other options.  Before running ASMLG, the options menu was used to select TRACE and CON options resulting in the instruction trace shown on the left window executing standard linkage followed by SVC 35 WTO to display the "hello world" message.  After installing z390 and J2RE runtime on a PC with Windows XP or 2000, you can run this demo yourself and use EDIT to view the MLC and generated BAL, OBJ, LST, and LOG files.  If you are on a Mac or Linux, join the z390 project and help port it.
