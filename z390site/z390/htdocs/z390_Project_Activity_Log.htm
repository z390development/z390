z390 Project Activity Log
Project z390
This is a log of the z390 Project activities starting with the move to the www.sourceforge.net/projects/z390 project web site. 
 
Date Description
10/10/07 The z390 source code for v1.3.08 is now on CVS at www.sourceforge.net.  To browse the source code online go to www.sourceforge.net./project/z390/z390.  The directories contain the following members:
  • bat - batch commands such as ASMLG.BAT stored in the z390 install directory
  • bmk - benchmark timing program and bat
  • cics - CICS compatible transaction manager programs, macros, and bats
  • demo - demo programs and bats
  • doc - documentation MS Word docs
  • guam - Graphical User Access Method demo and test programs and bats
  • linklib- utilitiy programs, called subroutine programs, and bats
  • mac - z390 standard macro library supported by z390 runtime svc's in sz390.java
  • pl - Linux perl scripts to support bat file execution stored in the install directory
  • soa - Service Oriented Architecture demo and test programs and bats
  • src - z390 J2SE 1.6.2 compatible Java source modules packaged into z390.jar in the install directory
  • vsam - VSAM access method demo and test programs and bats
  • vse - VSE compatible demo and test programs, macro library, and bats

This step took several days while I learned how to install and use the following:

  • puttygen.exe - utility for Windows to generate SSH private and public key
    • Generate SSH private and public key for use in accessing cvs respository
    • Upload public key to www.sourceforge.net using admin settings page
  • wincvs - CVS client with GUI interface for Windows
    • Read www.cvshome.org documentation
    • Create local repository and tested loading z390 v1.3.08 sources
    • Setup access to the z390 project repository on www.sourceforge.net
    • Import z390 v1.3.08 sources from Windows winscp client to www.sourceforge.net cvs
    • browse sources using web browser
10.09/07 Two new email groups hosted by www.sourceforge.net have been setup:
  • z390-news@sourceforge.net - news and release updates on z390
  • z390-discussion@sourceforge.net - open discussion and Q/A for z390 users and developers

 
10/06/07 The z390 project website was moved to z390.sourceforge.net using the winscp.sourceforge.net  tool for Windows which supports the SSH file transfer protocol used by www.sourceforge.net for both project website and project CVS version control for source code.  See PDF CVS manual here.
10/05/07 Application accepted and project links and space allocated
10/03/07 Application for z390 project hosting on www.sourceforge.net submitted
