V1.1.02 INSTTIME Instruction performance measurement program contributed by Melvyn Maltz.  Run BMK\INSTTIME.BAT to generate BMK\INSTTIME.OUT with statistics on all the instructions listed in BMK\INSTTIME.TXT.  V6 for v1.1.02 now has all the floating point instructions included.  Note this program requires SYSPARM set to BMK directory, ASCII, and MEM(5) options.
V1.1.02 BMKTIME Measure elapsed time from first to second call.  On the first call the program checks for the file named in SYSPARM.  If not found or if the file has more than one record, the file is rewritten with starting time.  On the second call if file has exactly 1 record with starting time, the file is rewritten with starting time, ending time, and elapsed time in milliseconds.  This program can be used to measure changes in any process run between calls to BMKTIME.   Run BMK\BMKTIME.BAT to see example timing for DEMO\DEMOM8Q1.MLC showing 40K vs 35K macro instructions per second on previous release.
