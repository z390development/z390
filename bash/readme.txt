readme.txt for Linux and Mac users of z390 as of 2021-04-01 Don Higgins don@higgins.net

The bash directory contains the following bash scripts:

  cmd.pl  BAT command processor
  dos.pl  DOS command processor
  z390.pl z390 command procesor to assemble, link, and execution z390 programs without using bat files

This perl directory has been added to the path for commannds issued by the z390.jar GUI interface.

For help with Linux and Mac use of these scripts with z390 please contact the volunteer contributor:

  John Ganci  jyganci@gmail.com
