z390 Graphical User Access Method GUAM demos and regression tests

All the programs in guam\demo and guam\test can be run usingthe
command rtguam.bat from the z390 install directory.  Each program
uses the z390 option GUAM to invoke the GUI user dialogue interface
which has 3 views - MCS console, TN3290 screen, and GUAM graphics.

Enter text on GUI command line when prompted for input.
To end program session enter END or press escape or PF3.

For more information on the z390 GUAM services see the GUAM User Guide
available on www.z390.org.
